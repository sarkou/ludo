backend :
    requetes http:  nodejs , express.
    serveur: gérer l'état du jeux, gérer le tour des joueurs
             gérer le jet des dés, gérer le mouvement  des pions

frontend: utilisé HTML CSS(SCSS) pour la maquette et Javascript pour communiquer
          avec le backend
dossier nodejs sera le backend et le dossier javascript sera le frontend

pour initialiser un projet : npm init -y
pour instaaller express : npm install express body-parser

pour exécuter le serveur : node app.js
